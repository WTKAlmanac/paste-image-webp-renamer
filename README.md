# Paste Image WebP Renamer

一个用于 Obsidian 的桌面端插件。

它会在你粘贴图片时自动完成以下处理：

- 将图片转换为 `WebP`
- 按“当前笔记路径 + 图片 CRC32”生成文件名
- 按 Obsidian 当前附件目录规则保存图片
- 如果转换后的图片大于 `1 MB`，弹出缩放界面，手动压缩到限制以内
- 粘贴完成后，自动在编辑器中插入对应的嵌入链接

## 功能说明

当你在 Obsidian 编辑器里粘贴图片时，插件会拦截默认粘贴流程，并改为：

1. 识别剪贴板中的图片文件
2. 将图片编码为 `WebP`
3. 以当前笔记路径为基础生成文件名前缀
4. 计算图片内容的 `CRC32`，追加到文件名中
5. 生成形如下面这样的文件名：

```text
你的笔记路径-CRC32.webp
```

例如当前笔记是：

```text
人物/魏国/曹操.md
```

那么生成的图片文件名大致会是：

```text
人物-魏国-曹操-1A2B3C4D.webp
```

如果目标路径下已经存在同名文件，插件不会重复写入，而是直接插入已有文件的嵌入链接。

## 大图处理逻辑

插件默认会先以固定质量将图片转换为 `WebP`。

如果转换结果超过 `1 MB`，会弹出一个压缩对话框：

- 自动计算一个尽量大的可用缩放比例
- 显示压缩后的预估分辨率与文件大小
- 你可以继续手动调整缩放比例
- 只有当结果不超过 `1 MB` 时，才允许确认粘贴

## 适用环境

- Obsidian `1.5.0` 及以上
- 仅支持桌面端

## 安装方式

将本仓库文件放入你的 Obsidian 插件目录：

```text
<你的库目录>/.obsidian/plugins/paste-image-webp-renamer/
```

至少需要包含：

- `manifest.json`
- `main.js`

然后在 Obsidian 中启用该插件。

## 使用方式

1. 打开任意 Markdown 笔记
2. 直接粘贴截图或剪贴板中的图片
3. 插件会自动保存图片并插入 `![[...]]` 引用
4. 如果图片过大，按提示调整缩放比例后确认

## 命名规则

图片文件名由两部分组成：

- 当前笔记路径（去掉 `.md` 后，将路径分隔符替换为 `-`）
- 图片二进制内容的 `CRC32` 校验值

这样做的好处是：

- 文件名可读，能看出图片来自哪篇笔记
- 相同内容的图片会得到稳定的哈希后缀
- 能减少重复保存同一张图片的情况
